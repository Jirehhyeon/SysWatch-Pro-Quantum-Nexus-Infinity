## 🎮 조작법

### 기본 조작
- **ESC** 또는 **Q** - 종료
- **SPACE** - 홀로그램 스크린샷 저장
- **R** - 퀀텀 유니버스 리셋

### 시스템 최적화 (NEW!)
- **1** - 메모리 캐시 정리
- **2** - 무거운 프로세스 제거
- **3** - 서비스 최적화
- **4** - 임시 파일 삭제

### 인터랙티브 기능
- **마우스 클릭** - 패널 선택/이동
- **프로세스 클릭** - 프로세스 선택 후 Kill
- **패널 드래그** - 3D 공간에서 패널 이동
